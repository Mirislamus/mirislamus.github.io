Мобильный телефон

Для погашения займа с помощью вашего мобильного телефона, убедитесь, что на вашем балансе есть достаточная для погашения сумма. Выберите свой вариант погашения займа со счета вашего мобильного телефона:

  • Заявка

    Отправьте SMS на короткий номер 7878 с указанием необходимой для оплаты займа суммы в формате wb 123456789 1000, где 123456789 - ваш логин в Вэббанкире, 1000 - сумма в рублях.

  • Подтверждение

    Дождитесь SMS с запросом подтверждения оплаты и указанием суммы, которая будет списана с вашего мобильного счета.

  • Ответ

    Для подтверждения оплаты, отправьте ответное сообщение, содержащее текст «1».

  • Отказ

    Если вы не готовы подтвердить оплату, отправьте ответное сообщение с текстом «0».

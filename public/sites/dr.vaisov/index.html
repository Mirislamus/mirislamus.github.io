ОБОРУДОВАНИЕ

Технологии мирового класса

Hollywood Spectra

Q-switched Nd:YAG лазер для пилинга, омоложения и лечения пигментации. Результат виден после первой процедуры.

DermaV

Многофункциональный лазер для удаления сосудов, папиллом, лечения сосудистых звёздочек и купероза.

LaseMD

Фракционный таллиевый лазер для омоложения, улучшения текстуры кожи и лечения мелазмы.

UltraSkin

HIFU-технология для нехирургической подтяжки лица и шеи без периода реабилитации.

Volnewmer

Монополярный RF-аппарат для подтяжки кожи век, лица и тела. Эффект лифтинга без операции.

Forever Young BBL

Широкополосный свет для омоложения, лечения акне, пигментации и сосудистых изменений.
